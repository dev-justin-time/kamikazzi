        04

        Linkup — shared realtime capture

        Sectional purpose: fuse multiple stereoscopic / multi-LiDAR / camera inputs across paired devices into one shared workload that assembles a 360° multi-stereogram, rendered in realtime 3D.

        Peer-to-peer device pairing & shared payloads use the Puter Peer API (puter.peer.serve() / connect() with TURN relays) so multiple phones + desktop stream capture into one collaborative scene without third-party signaling.

        05

        Code stack — best-tool-per-job

        Fastest performance, stable & secure. A shared Rust geometry core compiled to every platform, native mobile shells for camera/LiDAR access, and native desktop for master orchestration.
