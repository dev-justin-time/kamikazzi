# Going Balls - Web Edition

A 3D physics-based rolling platformer inspired by "Going Balls". 

## Features
- Physics-driven ball movement using `cannon-es`.
- Dynamic camera following the player.
- Level obstacles including pendulums, ramps, and narrow paths.
- Coin collection and progress tracking.
- Mobile-friendly controls (Virtual joystick via `nipplejs`).
- Keyboard controls (WASD/Arrows) for Desktop.

## Controls
- **WASD / Arrow Keys**: Steer and Accelerate.
- **Mouse Drag**: Click and drag to steer and move.
- **Mobile Joystick**: On-screen control for steering and speed.

## Technical Details
- Built with Three.js for rendering.
- Cannon-es for high-performance physics.
- Optimized for mobile viewports.
