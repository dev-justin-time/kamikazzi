How to use

  1. Right-click each preview and choose "Save image as...", OR click the download buttons below each canvas.
  2. Save icon-192.png to /assets/icon-192.png
  3. Save icon-512.png to /assets/icon-512.png
  4. Also save the 512px version as /assets/apple-touch-icon.png
  5. The SVG at /assets/icon.svg is already set up as the favicon.
